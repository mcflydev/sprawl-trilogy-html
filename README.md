# 🌌 Sprawl Trilogy - HTML Semântico

> "O céu sobre o porto tinha a cor de uma televisão sintonizada num canal fora do ar." — Neuromancer

Este projeto é um ambiente de estudos focado no desenvolvimento de uma página web estrutural sobre a **Trilogia do Sprawl** (*Neuromancer*, *Count Zero* e *Mona Lisa Overdrive*), clássicos do cyberpunk escritos por William Gibson.

O objetivo principal aqui é consolidar os fundamentos de **HTML5 Semântico**, garantindo acessibilidade e boa estruturação de dados antes de aplicar qualquer camada de estilo (CSS) ou comportamento (JavaScript).

---

## 🎯 Objetivos de Aprendizado

- [x] Configuração do ambiente de desenvolvimento (VS Code + Git/GitHub).
- [ ] Estruturação de conteúdo usando tags semânticas (`<header>`, `<main>`, `<article>`, `<section>`, `<footer>`).
- [ ] Uso correto de hierarquia de títulos (`<h1>` a `<h6>`).
- [ ] Criação de listas, links de navegação interna/externa e incorporação de imagens.
- [ ] Prática de versionamento de código com Git.

## 🛠️ Tecnologias Utilizadas

Nesta fase do projeto, o código é puramente estrutural:
*   **HTML5** (Semântico puro)

## 🚀 Como Visualizar o Projeto

Como o projeto utiliza apenas HTML puro, você pode visualizá-lo de duas formas:

1. **Localmente:** Clonando este repositório e abrindo o arquivo `index.html` direto no seu navegador (ou usando a extensão *Live Server* no VS Code).
2. **Código:** Navegando pelos arquivos diretamente aqui no GitHub.
